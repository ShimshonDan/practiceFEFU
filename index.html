Pet-Проекты

"Электронный фармаколог" - это инструмент для людей, связанных с медициной.

Настройка фильтров, вывод аналогов, важная информация о лекарственном препорате - это возможности, которое представляет наше приложение

Используемый стек:

  • Dart / Flutter - для ускорения разработки;
  • FireBase - база данных;
  • SQL - язык для работы с базой данных;

Используемый стек:

  • Python
